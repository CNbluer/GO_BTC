**v3**

已经更新到第三代版本,本次更新说明：

1、将区块加入到bolt数据库中，区块信息得以保存。

2、增加了POW挖矿系统，每一个区块通过哈希碰撞才能上链。

3、增加了迭代器，通过迭代器遍历整个区块。

3、增加了命令行控制系统，得以方便操作。

												——————2018.11.10
